# 🧠 NeuralCompile

An AI-powered IDE built with [Reflex](https://reflex.dev) that analyzes, compiles, and visualizes code using LLMs.

## Features

- 🖊️ **Monaco Editor** — VS Code-like editor in the browser
- 🔍 **Code Analyzer** — AI-powered static analysis and error detection
- 🌳 **AST Viewer** — Abstract Syntax Tree visualization
- 📊 **CFG Generator** — Control Flow Graph generation
- 🤖 **AI Suggester** — Intelligent code suggestions via Groq LLM
- 📜 **History** — Code execution and analysis history
- 🌐 **Multi-language** — Supports Python, JavaScript, C++, Java, and more

## Tech Stack

- **Frontend/Backend**: [Reflex](https://reflex.dev)
- **LLM**: Groq (LLaMA 3)
- **Database**: SQLite (via SQLModel)
- **Visualization**: NetworkX + Graphviz
